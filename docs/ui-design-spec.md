#### 3.2.1 正弦波曲线 (Sine Wave) — node `2:34`

尺寸: 840 × 140 px

由四层 SVG 矢量线叠加：

| 图层 | 节点 | 说明 | 颜色 | 透明度 | 线宽 |
|------|------|------|------|--------|------|
| Y 轴参考线 | `2:35` | 左侧竖线，暗示坐标轴 | `rgb(158, 255, 0)` | 12% | 1 px |
| X 轴参考线 | `2:36` | 横贯虚线，暗示 y=0 基线 | `rgb(158, 255, 0)` | 15% | 1 px |
| 外层辉光 | `2:37` | 4 周期三次贝塞尔曲线外光晕 | `rgb(158, 255, 0)` | 10% | 14 px |
| 主线 | `2:38` | 4 周期三次贝塞尔实线 | `rgb(158, 255, 0)` | 100% | 2.5 px |

> 曲线公式（三次贝塞尔逼近）：`y = A·sin(2π/T·x)`，振幅 100px，周期约 200px

SVG 路径控制点（4 周期，840px 宽）：

```
M 20,120 C 56,120  84,20   120,20   ← 周期 1
          C 156,20  184,120 220,120  ← 周期 2
          C 256,120 284,20  320,20   ← 周期 3
          C 356,20  384,120 420,120  ← 周期 4
```

## 八、跨端适配策略（Cocos Creator 固定宽高比）

Cocos Creator 采用固定宽高比等比缩放策略，设计基准 1920×1080 (16:9)。

| 场景 | 画布基准 | 适配方式 |
|------|----------|----------|
| PC (1920×1080) | 1:1 原始 | 点对点渲染 |
| PC (2560×1440) | 等比放大 | fitHeight / fitWidth |
| 移动端 (390×844) | 等比缩小 | 自动缩放，黑边填充 |

> 当前布局全部使用 flexbox 居中，在任意宽高比下元素自动收敛到容器中心，不需要额外断点适配。
